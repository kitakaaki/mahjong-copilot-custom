本儲存庫是上游專案的自訂分支。原始功能、安裝方式與完整文件請參考：

## Changes In This Fork / 本 Fork 的變更

- Default Majsoul URL changed to JP server (`https://game.mahjongsoul.com/`).
- Added Japanese UI language option (JA).
- Adjusted and fixed parts of auto-play logic (including click-coordinate and stability fixes).
- Added autoplay settings options for auto-hu enable/disable and result-based AI random level adjustment (with separate 3P/4P rank rules).
- Improved window/viewport behavior to reduce out-of-bounds issues at fixed resolutions.
- 預設雀魂網址改為日服（`https://game.mahjongsoul.com/`）。
- 新增日文介面語言選項（JA）。
- 調整並修正部分自動打牌邏輯（包含點擊座標與穩定性修正）。
- 新增自動打牌設定項目，可切換自動和了開關，以及依對局結果自動調整 AI 隨機等級（3人/4人採不同排名規則）。
- 優化視窗/viewport 行為，降低固定解析度下的越界問題。
